# 招 AI Native 全栈 / FDE

我们在做一个 AI 电商定制平台，方向包括电商、流量、供应链、金融风控、机器人和 Agentic workflow。

想找一类人：不是只会调用大模型 API，而是能用 AI-native 的方式，自己驱动完整 SDLC，把一个模糊业务问题做成可上线、可验证、可迭代的应用。

## 你会做什么

和领域专家一起，把电商/流量/供应/风控里的复杂问题，拆成产品、数据、规则、Agent、前后端和评估闭环。

具体可能包括：

- 多租户电商配置平台
- 商品/订单/优惠券/支付/分期/风控配置
- Agent 对话式配置助手
- 自进化决策引擎
- GenUI / Workflow / Skill / MCP
- 从 MVP 到上线验证的完整交付

## 我们希望你

- 会全栈开发，前后端都能上手
- 熟悉 AI coding，真正在用 AI 改变研发流程
- 做过 Agent / RAG / Workflow / Tool Calling 更好
- 有电商、流量、供应链、金融、SaaS、App、开源项目经验加分
- 能自驱，不等完整 PRD
- 有产品 taste，能做出业务真的愿意用的东西

## 不太适合

- 只会写 prompt
- 只会调 API
- 只等需求
- 不关心指标和上线效果
- 做完项目不沉淀流程
- 只写代码，不理解业务

我们更想找的是那种 builder：能理解业务，能写代码，能设计系统，也能把经验沉淀成可复用的 Skill 和工作流。

感兴趣可以带项目/作品来聊。
